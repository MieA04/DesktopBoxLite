## 1. 技术选型确认与理由

### 1.1 核心技术栈

| 层面 | 技术选择 | 选择理由 |
|------|---------|---------|
| 桌面框架 | Tauri 2 | 相比 Electron，Tauri 2 应用体积小 10-20 倍（< 5MB），内存占用低 5-10 倍；使用操作系统原生 WebView2（Windows 10+ 内置），无需捆绑 Chromium；Rust 后端提供零开销抽象和系统级 API 调用能力 |
| 前端语言 | TypeScript (vanilla) | 项目规模较小（5 个组件），引入 React/Vue 等框架会增加约 50KB+ 的 JS 体积和构建复杂度；vanilla TS 配合模块化类结构足以支撑，且符合"极简轻量"的设计原则 |
| UI 渲染 | 原生 HTML + CSS | 无需虚拟 DOM 和复杂状态管理，CSS Grid 原生支持响应式布局，DOM 操作量级小（最多 200+ 图标节点），原生 API 性能足够 |
| 后端语言 | Rust (稳定版) | Tauri 2 原生后端语言；零成本抽象保证系统调用性能（Windows API、文件扫描）；内存安全无 GC，适合长时间运行的后台进程 |
| 构建工具 | Vite (Tauri 2 内置) | Tauri 2 create 模板默认集成 Vite；开发时 HMR（热模块替换）速度极快；生产构建时 Tree Shaking 压缩 JS 体积 |

### 1.2 Rust 关键依赖选型

| Crate | 版本 | 理由 |
|-------|------|------|
| `serde` / `serde_json` | 1.x | Rust 生态事实标准的序列化框架，支持 derive 宏，零开销反序列化；用于 config.json 解析和数据传输 |
| `tauri-plugin-global-hotkey` | 2.x | Tauri 2 官方插件，与 Tauri API 深度集成，支持注册/注销全局快捷键回调；版本锁定避免 API 变更 |
| `opener` | 0.7+ | 跨平台打开文件/URL，内部调用 `ShellExecuteW`（Windows），仅 200+ 行代码的轻量库 |
| `log` + `env_logger` + `chrono` | 4.x / 0.11 / 0.4 | `log` 是 Rust 日志门面（facade）；`env_logger` 支持 RUST_LOG 环境变量控制日志级别；`chrono` 用于日志文件名时间戳格式化 |
| `windows-sys` | 0.52+ | 官方 Windows API 绑定，零开销 FFI 调用；用于 `FindWindowW`、`ShowWindow`、注册表操作等；仅为 Windows 平台条件编译 |
| `dirs` | 5.x | 跨平台获取标准目录（config_dir、home_dir），用于定位 `%APPDATA%/DesktopBoxLite` 和桌面路径 |

### 1.3 前端依赖选型

| 依赖 | 版本 | 理由 |
|------|------|------|
| `typescript` | 5.x | 类型检查，编译到 ES2020；严格模式防止隐式 `any` |
| `@tauri-apps/api` | 2.x | Tauri 2 前端 JS API，提供 `invoke`、`window` 等核心调用 |
| `@tauri-apps/plugin-global-hotkey` | 2.x | 前端侧全局快捷键事件监听（可选，主要用于前端自定义监听） |

### 1.4 否决的技术方案

| 方案 | 否决理由 |
|------|---------|
| Electron | 内存占用 200MB+，安装包 150MB+，与"极简轻量"原则冲突 |
| React / Vue | 200+ 图标规模下虚拟 DOM 优化收益甚微；增加构建体积和加载时间；team 经验偏向原生 |
| Tauri v1 | Tauri 2 已稳定，API 更现代、插件系统更完善；v1 已进入维护模式 |
| `winapi` crate | `windows-sys` 是微软官方维护的 Rust Windows API 绑定，类型更安全，更新更及时 |
| `notify` crate 文件监听 | P0 阶段采用轮询（500ms interval）更简单可靠；P2 时可按需升级 |

### 2.2 目录设计原则

- **前端组件化**：每个 `.ts` 文件对应一个独立 UI 组件，通过类封装 DOM 创建、事件绑定、状态管理。
- **后端模块化**：每个 `.rs` 文件对应一组单一职责的功能集合，通过 `lib.rs` 统一导出和命令注册。
- **类型共享**：前端和后端的类型定义分离（前端 `types/index.ts`，后端 Rust struct），通过文档保证字段一致。
- **工具分离**：纯逻辑工具函数放在 `utils/`，UI 组件放在 `components/`。

### 3.3 executor.rs — 指令执行

```rust
// 职责：执行系统指令/脚本，不显示控制台窗口

use std::process::Command;
use log::{info, error};

/// 执行系统指令
/// 参数 cmd: 完整的命令字符串，如 "calc.exe" 或 "cmd.exe /c echo hello"
/// 成功返回 Ok(()), 失败返回 Err(错误描述)
pub fn execute_command(cmd: &str) -> Result<(), String> {
    // 使用 cmd.exe /c 包装（对 .exe 直接执行）
    // 关键：设置 CREATE_NO_WINDOW 标志避免控制台弹窗
    let output = Command::new("cmd")
        .args(["/C", cmd])
        .creation_flags(0x08000000) // CREATE_NO_WINDOW
        .output()
        .map_err(|e| format!("执行失败: {}", e))?;

    if !output.status.success() {
        let stderr = String::from_utf8_lossy(&output.stderr);
        error!("指令执行失败: {} (错误: {})", cmd, stderr);
    } else {
        info!("指令执行成功: {}", cmd);
    }
    Ok(())
}
```

**注意事项**：
- `creation_flags(0x08000000)` 需要 Rust std 的 `CommandExt` trait（Windows 平台）
- 指令执行结果只写日志，永不弹窗（符合需求文档）
- 对于 `.bat` / `.ps1` 脚本，同样通过 `cmd.exe /C` 执行
- 使用 `std::process::Stdio::piped()` 避免子进程继承控制台

```rust
// 隐藏桌面图标核心伪代码
pub fn hide_desktop_icons() -> Result<(), String> {
    unsafe {
        // 1. 找到 Progman 窗口
        let progman = FindWindowW(w!("Progman"), None);
        if progman == 0 {
            return Err("找不到 Progman 窗口".into());
        }

        // 2. 尝试直接找到 SysListView32（Windows 10 常用路径）
        let mut hwnd_shell = FindWindowExW(progman, None, w!("SHELLDLL_DefView"), None);
        if hwnd_shell == 0 {
            // 备用路径：WorkerW 窗口模式（某些 Windows 版本）
            // 遍历所有 WorkerW 窗口查找 SHELLDLL_DefView
            let mut hwnd_workerw = FindWindowExW(None, None, w!("WorkerW"), None);
            while hwnd_workerw != 0 {
                hwnd_shell = FindWindowExW(hwnd_workerw, None, w!("SHELLDLL_DefView"), None);
                if hwnd_shell != 0 { break; }
                hwnd_workerw = FindWindowExW(None, hwnd_workerw, w!("WorkerW"), None);
            }
        }

        // 3. 在 SHELLDLL_DefView 中找到 SysListView32
        if hwnd_shell == 0 {
            return Err("找不到 SHELLDLL_DefView 窗口".into());
        }
        let hwnd_list = FindWindowExW(hwnd_shell, None, w!("SysListView32"), None);
        if hwnd_list == 0 {
            return Err("找不到 SysListView32 窗口".into());
        }

        // 4. 隐藏图标列表
        ShowWindow(hwnd_list, SW_HIDE);
        info!("桌面图标已隐藏");
        Ok(())
    }
}

// 恢复桌面图标类似，使用 SW_SHOW 替代 SW_HIDE
```

**`w!()` 宏**：`windows-sys` crate 提供的宽字符串宏，等效于 `L"..."`。

**支持的图标文件类型**：

| 扩展名 | 处理方式 |
|--------|---------|
| `.lnk` | 解析快捷方式目标路径，名称去 `.lnk` 后缀 |

**图标提取方案**：

由于从 `.lnk` 提取图标较为复杂（需要 `IShellLink` COM 接口），P0 阶段使用简化方案：

1. 对于 `.exe` 文件，直接使用其路径作为 `icon_path`，前端通过 `https://icon.horse/icon/tld` 或 `file://` 协议加载
2. 对于 `.lnk` 文件，先解析出目标 exe 路径，再使用该 exe 的图标
3. 对于 `.url` 文件，使用默认的 URL 图标
4. 回退方案：所有图标统一使用系统默认图标

**可行方案**（按优先级排序）：
1. **Tauri custom protocol**：注册自定义 asset 协议，后端通过 Windows API 提取图标后以 Base64 或文件流形式返回
2. **前端文件协议**：使用 `convertFileSrc` (Tauri 2) 将本地路径转换为 WebView 可访问的 URL
3. **Base64 内嵌**：后端提取图标后转为 Base64 data URL，传输给前端直接渲染

推荐方案：使用 Tauri 2 的 `asset` 协议或 `convertFileSrc`，将桌面快捷方式路径转换为 WebView 可加载的资源 URL。

**实现方案**：

由于 `env_logger` 不原生支持文件输出，有以下两种方案：

**方案 A（推荐，P0 方案）**：`env_logger` 输出到 stderr + 自己实现文件写入

```rust
use log::{Level, Log, Metadata, Record, SetLoggerError};
use std::fs::{OpenOptions, create_dir_all};
use std::io::Write;
use std::sync::Mutex;

struct FileLogger {
    file: Mutex<std::fs::File>,
    level: LevelFilter,
}

impl Log for FileLogger {
    fn enabled(&self, metadata: &Metadata) -> bool {
        metadata.level() <= self.level
    }
    fn log(&self, record: &Record) {
        if self.enabled(record.metadata()) {
            let timestamp = Local::now().format("%Y-%m-%d %H:%M:%S");
            let msg = format!(
                "[{}] [{}] {}: {}\n",
                timestamp,
                record.level(),
                record.target(),
                record.args()
            );
            if let Ok(mut file) = self.file.lock() {
                let _ = file.write_all(msg.as_bytes());
                let _ = file.flush();
            }
        }
    }
    fn flush(&self) {
        if let Ok(mut file) = self.file.lock() {
            let _ = file.flush();
        }
    }
}
```

**方案 B（更完善的方案）**：使用 `log4rs` crate（不额外引入，保持依赖最小化）。

**P0 采用方案 A**，文件路径 `./logs/YYYY-MM-DD.log`，启动时自动创建 `logs/` 目录。

```rust
use winreg::enums::*;

pub fn set_auto_start(enabled: bool) -> Result<(), String> {
    let hkcu = RegKey::predef(HKEY_CURRENT_USER);
    let run_path = r"Software\Microsoft\Windows\CurrentVersion\Run";
    let key = hkcu.open_subkey_with_flags(run_path, KEY_SET_VALUE)
        .map_err(|e| format!("打开注册表失败: {}", e))?;

    if enabled {

        key.set_value("DesktopBoxLite", &exe_path.to_string_lossy().to_string())
            .map_err(|e| format!("写入注册表失败: {}", e))?;
    } else {
        key.delete_value("DesktopBoxLite")
            .map_err(|e| format!("删除注册表项失败: {}", e))?;
```

### 5.9 utils/iconParser.ts — 图标名称处理

```typescript
/** 要去除的文件扩展名列表 */
const EXTENSIONS_TO_STRIP = ['.lnk', '.exe', '.url', '.txt', '.doc', '.docx',
  '.xls', '.xlsx', '.ppt', '.pptx', '.pdf', '.zip', '.rar', '.7z',
  '.png', '.jpg', '.jpeg', '.gif', '.bmp', '.mp3', '.mp4', '.avi'];

/**
 * 从文件名去除常见扩展名
 * 例如: "计算器.lnk" -> "计算器"
 *       "note.exe" -> "note"
 *       "我的文档" -> "我的文档"（没有扩展名时保持不变）
 */
export function trimFileExtension(filename: string): string {
  for (const ext of EXTENSIONS_TO_STRIP) {
    if (filename.toLowerCase().endsWith(ext)) {
      return filename.slice(0, -ext.length);
    }
  }
  // 处理两次扩展名情况（如 .tar.gz）
  const dotIndex = filename.lastIndexOf('.');
  if (dotIndex > 0) {
    return filename.slice(0, dotIndex);
  }
  return filename;
}

/**
 * 过滤图标列表（不区分大小写，模糊匹配名称）
 */
export function filterIcons(icons: IconData[], query: string): IconData[] {
  if (!query.trim()) return icons;
  const lower = query.toLowerCase();
  return icons.filter(icon => icon.name.toLowerCase().includes(lower));
}
```

**ResizableWindow.ts** — 拖拽缩放实现：

```typescript
import { ResizeDirection } from '../types';

export class ResizableWindow {
  public element: HTMLElement;
  private isResizing = false;
  private startX = 0;
  private startY = 0;
  private startWidth = 0;
  private startHeight = 0;
  private direction: ResizeDirection | null = null;
  private onResizeEndCallback: ((w: number, h: number) => void) | null = null;
  private boundOnMouseMove: (e: MouseEvent) => void;
  private boundOnMouseUp: (e: MouseEvent) => void;

  static readonly MIN_WIDTH = 120;
  static readonly MIN_HEIGHT = 150;

  constructor(parentId: string, initialWidth: number, initialHeight: number) {
    // 创建主容器
    this.element = document.createElement('div');
    this.element.className = 'resizable-window';
    this.element.style.width = `${initialWidth}px`;
    this.element.style.height = `${initialHeight}px`;

    document.getElementById(parentId)!.appendChild(this.element);

    // 绑定事件处理器（使用箭头函数保持 this 指向）
    this.boundOnMouseMove = this.onMouseMove.bind(this);
    this.boundOnMouseUp = this.onMouseUp.bind(this);

    // 创建所有方向的手柄
    this.createResizeHandles();
  }

  private createResizeHandles(): void {
    // 创建 8 个方向手柄
    const directions = [
      ResizeDirection.N, ResizeDirection.S, ResizeDirection.W, ResizeDirection.E,
      ResizeDirection.NW, ResizeDirection.NE, ResizeDirection.SW, ResizeDirection.SE,
    ];

    for (const dir of directions) {
      const handle = document.createElement('div');
      handle.className = `resize-handle resize-handle--${dir}`;
      handle.addEventListener('mousedown', (e) => this.startResize(e, dir));
      this.element.appendChild(handle);
    }
  }

  private startResize(e: MouseEvent, direction: ResizeDirection): void {
    e.preventDefault();
    this.isResizing = true;
    this.direction = direction;
    this.startX = e.clientX;
    this.startY = e.clientY;
    this.startWidth = this.element.offsetWidth;
    this.startHeight = this.element.offsetHeight;

    document.addEventListener('mousemove', this.boundOnMouseMove);
    document.addEventListener('mouseup', this.boundOnMouseUp);
  }

  private onMouseMove(e: MouseEvent): void {
    if (!this.isResizing || !this.direction) return;

    const dx = e.clientX - this.startX;
    const dy = e.clientY - this.startY;

    let newWidth = this.startWidth;
    let newHeight = this.startHeight;

    // 根据方向计算新尺寸
    const dir = this.direction;
    // 水平方向
    if (dir.includes('e')) newWidth = this.startWidth + dx;
    if (dir.includes('w')) newWidth = this.startWidth - dx;
    // 垂直方向
    if (dir.includes('s')) newHeight = this.startHeight + dy;
    if (dir.includes('n')) newHeight = this.startHeight - dy;

    // 应用最小尺寸限制
    newWidth = Math.max(newWidth, ResizableWindow.MIN_WIDTH);
    newHeight = Math.max(newHeight, ResizableWindow.MIN_HEIGHT);

    this.element.style.width = `${newWidth}px`;
    this.element.style.height = `${newHeight}px`;
  }

  private onMouseUp(_e: MouseEvent): void {
    if (!this.isResizing) return;
    this.isResizing = false;
    this.direction = null;

    document.removeEventListener('mousemove', this.boundOnMouseMove);
    document.removeEventListener('mouseup', this.boundOnMouseUp);

    // 触发缩放结束回调（持久化尺寸）
    if (this.onResizeEndCallback) {
      this.onResizeEndCallback(
        this.element.offsetWidth,
        this.element.offsetHeight
      );
    }
  }

  public onResizeEnd(callback: (w: number, h: number) => void): void {
    this.onResizeEndCallback = callback;
  }

  public getSize(): { width: number; height: number } {
    return {
      width: this.element.offsetWidth,
      height: this.element.offsetHeight,
    };
  }
}
```

```rust
#[derive(Debug, Clone, Serialize)]
pub struct DesktopIcon {
    pub name: String,
    pub path: String,
    pub icon_path: String,
}

/// 获取用户桌面路径
pub fn user_desktop_path() -> PathBuf {
    dirs::desktop_dir().unwrap_or_else(|| PathBuf::from("C:\\Users\\Public\\Desktop"))
}

/// 获取公共桌面路径
pub fn public_desktop_path() -> PathBuf {
    PathBuf::from("C:\\Users\\Public\\Desktop")

/// 扫描两个桌面目录，合并去重
pub fn scan_desktop_icons() -> Vec<DesktopIcon> {
    let mut all_icons = Vec::new();
    let mut seen = HashSet::new();

    // 扫描用户桌面
    let user_path = user_desktop_path();
    if let Ok(entries) = fs::read_dir(&user_path) {
        for entry in entries.flatten() {
            let path = entry.path();
            if let Some(icon) = file_to_icon(&path) {
                let key = format!("{}|{}", icon.name.to_lowercase(), icon.path.to_lowercase());
                if seen.insert(key) {
                    all_icons.push(icon);
                }
            }
        }
    }

    // 扫描公共桌面
    let pub_path = public_desktop_path();
    if pub_path != user_path {
        if let Ok(entries) = fs::read_dir(&pub_path) {
            for entry in entries.flatten() {
                let path = entry.path();
                if let Some(icon) = file_to_icon(&path) {
                    let key = format!("{}|{}", icon.name.to_lowercase(), icon.path.to_lowercase());
                    if seen.insert(key) {
                        all_icons.push(icon);
                    }
                }
            }
        }
    }

    // 按名称排序
    all_icons.sort_by(|a, b| a.name.to_lowercase().cmp(&b.name.to_lowercase()));

    info!("扫描完成: {} 个图标 (用户桌面 + 公共桌面)", all_icons.len());
    debug!("图标列表: {:?}", all_icons.iter().map(|i| &i.name).collect::<Vec<_>>());

    all_icons
}

/// 将文件路径转换为 DesktopIcon，仅处理可显示的文件
fn file_to_icon(path: &PathBuf) -> Option<DesktopIcon> {
    // 跳过隐藏文件和系统文件
    if let Some(name) = path.file_name() {
        let name_str = name.to_string_lossy();
        if name_str.starts_with('.') || name_str == "desktop.ini" || name_str == "Thumbs.db" {
            return None;
        }
    }

    // 仅显示文件（非目录）或快捷方式
    if !path.is_file() && !path.is_symlink() {
        return None;
    }

    let raw_name = path.file_stem()
        .unwrap_or_default()
        .to_string_lossy()
        .to_string();

    let abs_path = path.to_string_lossy().to_string();

    // icon_path: 使用 Tauri 的 asset 协议路径，或直接用 file://
    // 简单方案：直接使用文件路径，前端通过 convertFileSrc 转换
    let icon_path = abs_path.clone();

    Some(DesktopIcon {
        name: raw_name,
        path: abs_path,
        icon_path,
    })
}
```

```typescript
export function trimFileExtension(filename: string): string {

    if (lower.endsWith(ext)) {
```

```rust
use windows_sys::Win32::UI::WindowsAndMessaging::*;
use windows_sys::Win32::Foundation::HWND;
use log::{info, error};

/// Windows 宽字符串宏
macro_rules! w {
    ($s:expr) => {{
        let v: Vec<u16> = ($s.encode_utf16().chain(std::iter::once(0))).collect();
        v.as_ptr()
    }};
}

/// 隐藏桌面图标
pub fn hide_desktop_icons() -> Result<(), String> {
    unsafe {
        let progman = FindWindowW(w!("Progman"), std::ptr::null());
        if progman == 0 {
            return Err("未找到 Progman 窗口".into());
        }

        // 查找 SHELLDLL_DefView（直接子窗口）
        let mut shell_view = FindWindowExW(progman, 0, w!("SHELLDLL_DefView"), std::ptr::null());

        // 如果没找到，遍历 WorkerW 窗口（Windows 10/11 的另一种布局）
        if shell_view == 0 {
            let mut worker_w = FindWindowExW(0, 0, w!("WorkerW"), std::ptr::null());
            while worker_w != 0 {
                shell_view = FindWindowExW(worker_w, 0, w!("SHELLDLL_DefView"), std::ptr::null());
                if shell_view != 0 {
                    break;
                }
                worker_w = FindWindowExW(0, worker_w, w!("WorkerW"), std::ptr::null());
            }
        }

        if shell_view == 0 {
            return Err("未找到 SHELLDLL_DefView 窗口".into());
        }

        // 在 SHELLDLL_DefView 中找到 SysListView32
        let list_view = FindWindowExW(shell_view, 0, w!("SysListView32"), std::ptr::null());
        if list_view == 0 {
            return Err("未找到 SysListView32 窗口".into());
        }

        // 隐藏图标列表
        ShowWindow(list_view, SW_HIDE);
        info!("桌面图标已隐藏");
        Ok(())
    }
}

/// 恢复桌面图标
pub fn show_desktop_icons() -> Result<(), String> {
    unsafe {
        let progman = FindWindowW(w!("Progman"), std::ptr::null());
        if progman == 0 {
            return Err("未找到 Progman 窗口".into());
        }

        let mut shell_view = FindWindowExW(progman, 0, w!("SHELLDLL_DefView"), std::ptr::null());
        if shell_view == 0 {
            let mut worker_w = FindWindowExW(0, 0, w!("WorkerW"), std::ptr::null());
            while worker_w != 0 {
                shell_view = FindWindowExW(worker_w, 0, w!("SHELLDLL_DefView"), std::ptr::null());
                if shell_view != 0 { break; }
                worker_w = FindWindowExW(0, worker_w, w!("WorkerW"), std::ptr::null());
            }
        }

        if shell_view == 0 {
            return Err("未找到 SHELLDLL_DefView 窗口".into());

        let list_view = FindWindowExW(shell_view, 0, w!("SysListView32"), std::ptr::null());
        if list_view == 0 {
            return Err("未找到 SysListView32 窗口".into());
        }

        ShowWindow(list_view, SW_SHOW);
        info!("桌面图标已恢复");
        Ok(())
    }
```

```rust
use std::process::{Command, Stdio};
use log::{info, error};

#[cfg(target_os = "windows")]
use std::os::windows::process::CommandExt;

const CREATE_NO_WINDOW: u32 = 0x08000000;

/// 执行系统指令（不显示控制台窗口）
pub fn execute_command(cmd: &str) -> Result<(), String> {
    info!("执行指令: {}", cmd);

    let output = Command::new("cmd")
        .args(["/C", cmd])
        .creation_flags(CREATE_NO_WINDOW)
        .stdout(Stdio::piped())
        .stderr(Stdio::piped())
        .output()
        .map_err(|e| format!("执行失败: {}", e))?;

    if !output.status.success() {
        let stderr = String::from_utf8_lossy(&output.stderr);
        error!("指令执行失败 (退出码: {:?}): {}", output.status.code(), stderr);
    } else {
        info!("指令执行成功: {}", cmd);
    }

    Ok(())
}
```

```rust
        }
        return default_config;
    }

    match fs::read_to_string(&path) {
        Ok(content) => {
            match serde_json::from_str::<Config>(&content) {
                Ok(config) => {
                    info!("配置文件加载成功: {:?}", path);
                    config
                }
                Err(e) => {
                    error!("配置文件解析失败 ({}), 使用默认配置: {}", e, path.display());
                    Config::default()
                }
            }
        }
        Err(e) => {
            error!("读取配置文件失败 ({}), 使用默认配置: {:?}", e, path);
            Config::default()
        }
    }
}

/// 保存配置到文件
pub fn save_config(config: &Config) -> Result<(), String> {
    let path = config_path();
    ensure_config_dir().map_err(|e| format!("创建配置目录失败: {}", e))?;

    let json = serde_json::to_string_pretty(config)
        .map_err(|e| format!("配置序列化失败: {}", e))?;

    fs::write(&path, &json)
        .map_err(|e| format!("写入配置文件失败: {}", e))?;

    info!("配置已保存: {:?}", path);
    Ok(())
}

/// 保存窗口尺寸（仅修改窗口尺寸，保持其他字段不变）
pub fn save_window_size(width: u32, height: u32) -> Result<(), String> {
    let mut config = load_config();
    config.behavior.window_width = width;
    config.behavior.window_height = height;
    save_config(&config)?;
    info!("窗口尺寸已保存: {}x{}", width, height);
    Ok(())
}
```

```rust
use log::{Level, LevelFilter, Log, Metadata, Record, SetLoggerError};
use std::fs::{self, OpenOptions, create_dir_all};
use std::io::Write;
use std::path::PathBuf;
use std::sync::Mutex;
use chrono::Local;

struct FileLogger {
    file: Mutex<std::fs::File>,
    level: LevelFilter,
}

impl Log for FileLogger {
    fn enabled(&self, metadata: &Metadata) -> bool {
        metadata.level() <= self.level
    }

    fn log(&self, record: &Record) {
        if !self.enabled(record.metadata()) {
            return;
        }

        let timestamp = Local::now().format("%Y-%m-%d %H:%M:%S%.3f");
        let msg = format!(
            "[{}] [{}] {}: {}\n",
            timestamp,
            record.level(),
            record.target(),
            record.args()
        );

        // 同时输出到 stderr（开发时可在控制台看到）
        eprint!("{}", msg);

        // 写入文件
        if let Ok(mut file) = self.file.lock() {
            let _ = file.write_all(msg.as_bytes());
            let _ = file.flush();
        }
    }

    fn flush(&self) {
        if let Ok(mut file) = self.file.lock() {
            let _ = file.flush();
        }
    }
}

/// 获取当天日志文件路径
fn log_file_path() -> PathBuf {
    let log_dir = PathBuf::from("logs");
    let date_str = Local::now().format("%Y-%m-%d");
    log_dir.join(format!("{}.log", date_str))
}

/// 初始化日志系统
pub fn init_logging() -> Result<(), Box<dyn std::error::Error>> {
    // 创建 logs 目录
    let log_dir = PathBuf::from("logs");
    create_dir_all(&log_dir)?;

    // 打开日志文件（追加模式）
    let log_path = log_file_path();
    let file = OpenOptions::new()
        .create(true)
        .append(true)
        .open(&log_path)?;

    // 日志级别：优先使用 RUST_LOG 环境变量，默认 info
    let level = match std::env::var("RUST_LOG") {
        Ok(val) => match val.to_lowercase().as_str() {
            "error" => LevelFilter::Error,
            "warn" => LevelFilter::Warn,
            "info" => LevelFilter::Info,
            "debug" => LevelFilter::Debug,
            "trace" => LevelFilter::Trace,
            _ => LevelFilter::Info,
        },
        Err(_) => LevelFilter::Info,
    };

    let logger = FileLogger {
        file: Mutex::new(file),
        level,
    };

    log::set_boxed_logger(Box::new(logger))?;
    log::set_max_level(level);

    log::info!("日志系统初始化完成 (级别: {})", level);
    log::info!("日志文件: {}", log_path.display());

    Ok(())
}
```

```rust
use tauri::{

use log::info;

pub fn create_tray(app: &AppHandle) -> Result<(), Box<dyn std::error::Error>> {
    // 创建菜单项
    let toggle = MenuItemBuilder::with_id("toggle", "显示/隐藏").build(app)?;
    let reload = MenuItemBuilder::with_id("reload", "重载配置").build(app)?;
    let quit = MenuItemBuilder::with_id("quit", "退出").build(app)?;

    // 创建菜单
    let menu = MenuBuilder::new(app)
        .item(&toggle)
        .item(&reload)
        .separator()
        .item(&quit)
        .build()?;

    // 创建托盘图标
    TrayIconBuilder::new()
        .icon(app.default_window_icon().unwrap().clone())
        .menu(&menu)
        .tooltip("DesktopBox Lite")
        .on_menu_event(|app, event| {
            match event.id().as_ref() {
                "toggle" => {
                    if let Some(window) = app.get_webview_window("main") {
                        let visible = window.is_visible().unwrap_or(true);
                        let _ = if visible { window.hide() } else { window.show() };
                    }
                }
                "reload" => {
                    info!("重载配置...");
                    // 重新加载配置和注册快捷键
                    let config = crate::config::load_config();
                    let _ = crate::hotkey::unregister_hotkeys(app);
                    let _ = crate::hotkey::register_hotkeys(app, &config);
                }
                "quit" => {
                    // 退出前恢复桌面图标
                    let _ = crate::desktop::show_desktop_icons();
                    app.exit(0);
                }
                _ => {}
            }
        })
        .on_tray_icon_event(|tray, event| {
            if let TrayIconEvent::Click {
                button: MouseButton::Left,
                button_state: MouseButtonState::Up,
                ..
            } = event {
                // 左键单击切换显隐
                if let Some(app) = tray.app_handle() {
                    if let Some(window) = app.get_webview_window("main") {
                        let visible = window.is_visible().unwrap_or(true);
                        let _ = if visible { window.hide() } else { window.show() };
                    }
                }
            }
        })
        .build(app)?;

    info!("系统托盘图标已创建");
    Ok(())
}
```

```rust
use log::info;
use winreg::RegKey;
use winreg::enums::*;

/// 设置/取消开机自启动
pub fn set_auto_start(enabled: bool) -> Result<(), String> {
    let hkcu = RegKey::predef(HKEY_CURRENT_USER);
    let run_path = r"Software\Microsoft\Windows\CurrentVersion\Run";

    let key = hkcu.open_subkey_with_flags(run_path, KEY_SET_VALUE | KEY_QUERY_VALUE)
        .map_err(|e| format!("打开注册表失败: {}", e))?;

    if enabled {
        let exe_path = std::env::current_exe()
            .map_err(|e| format!("获取程序路径失败: {}", e))?;
        let path_str = exe_path.to_string_lossy().to_string();
        key.set_value("DesktopBoxLite", &path_str)
            .map_err(|e| format!("写入注册表失败: {}", e))?;
        info!("开机自启动已启用: {}", path_str);
    } else {
        match key.delete_value("DesktopBoxLite") {
            Ok(_) => info!("开机自启动已禁用"),
            Err(_) => { /* 值不存在，忽略 */ }
        }
    }

    Ok(())
}
```
