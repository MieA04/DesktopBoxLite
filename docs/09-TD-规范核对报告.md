### 2.3 权限声明

`capabilities/default.json` 中声明的权限与实际使用的 API 对照：

| 权限 | 使用位置 | 是否需要 |
|------|---------|---------|
| `core:default` | Tauri 基础 | ✅ |
| `core:window:allow-show` | `window.show()` | ✅ |
| `core:window:allow-hide` | `finish_hide` | ✅ |
| `core:window:allow-set-size` | 窗口缩放 | ✅ |

| `core:window:allow-close` | 退出 | ✅ |
| `core:window:allow-center` | 居中（当前未使用） | ⚠️ 可移除但无害 |
| `core:event:allow-listen` | `listen()` | ✅ |
| `core:event:allow-emit` | `app.emit()` | ✅ |
| `global-shortcut:default` | 快捷键插件 | ✅ |

### 2.4 数据流完整性

```
启动 → load_config() → 注册快捷键 → 创建托盘
  ↓
get_icons → 前端渲染图标网格
  ↓
用户交互 → 单击图标 → open_file (opener crate)
         → 拖拽缩放 → save_window_size
         → 搜索输入 → filterIcons (前端过滤)
         → 快捷键 → on_shortcut handler
              → toggle (Pressed only) → emit animate-hide/animate-show
              → custom command → execute_command
         → 托盘菜单 → toggle/reload/quit
```
